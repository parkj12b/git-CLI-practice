Common HTML Tags

if you didn't know them already...

Tag Name Description
HTML Html Tag that surrounds the whole html document
HEAD Head Head of the html file. Used to put meta and links
BODY Body Body of the html file. Everything except things in head goes in here
TITLE Title Title of the html file or the webpage. This is displayed on the tab of the web application
H1... Head 1 First-level heading. Should be used to identifying largeset text/title in the webpage
P Paragraph Used to surround paragraphs
BR Line Break Adds a blank line
A A tag Used to put hyperlink on a text, image, and more
IMG image tag used to insert image
B Bold Make the surrounded text bold
I Italics Make the surrounded text italics
TABLE Table Tag used to create and surround all the component of table
SPAN span used when part of the html has to be tagged and specified by class or id
